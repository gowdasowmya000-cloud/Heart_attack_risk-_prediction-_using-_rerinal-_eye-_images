# Heart Attack Risk Prediction using Retinal Eye Images

## Description

This project aims to predict the risk of heart attack using retinal eye images. By analyzing patterns in eye images, the system helps in early detection of cardiovascular risks.

## Features

* Image-based prediction system
* Early risk detection of heart attack
* Machine learning model integration
* User-friendly interface

## Tech Stack

* Python
* Machine Learning
* Deep Learning
* fuzzy c-means clustering
